Honest · where it breaks

Four traps I fell into.
So you don’t.

01
Tool churn
Vendors deprecate. Stay in plain files and standard formats. Skills travel.
02
Hallucination on facts
Models guess on long-tail data. Force citations. Verify what’s load-bearing.
03
GDPR and privacy edges
Client data needs scoped tools. Build the privacy fence before the workflow.
04
Mindset, not tech
7/10 digitisation projects fail on people. Adoption is the moat, not the model.
What this means for an agency like yours

Don’t perfect the setup.
Ship the first workflow this week.

01
Find the drain

Which repetitive task burns more than 5h a week on your team? Account brief? Client recap? Asset routing?

02
Wrap one workflow

Memory + tools + a 10-line prompt. Don’t build a platform. Build a habit.

03
One twin per role

Account Manager twin. Creative twin. Ops twin. Don’t try to make one for everyone.

04
Kill it if ROI is not there in 1 week

If it doesn’t save real time inside seven days, the workflow is wrong, not the tech.
