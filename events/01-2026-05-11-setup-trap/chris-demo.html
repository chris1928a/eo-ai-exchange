Operator to Architect.The hustle that felt like an asset was the liability the whole time.

Tool 06
Thin Harness
Claude Code.
The harness that runs the model in a loop. Reads files, manages context, enforces safety. ~50 line CLAUDE.md, not 20.000.
"My CLAUDE.md was 20.000 lines. The model's attention degraded. The fix was 200 lines, just pointers to documents." — Garry Tan
My harness layer
CLAUDE.md
Resolver. ~50 lines. Voice rules, anti-patterns, where to find what. Pointers, not content.
always loaded
ripgrep + glob + Read
Agentic search. No vector-DB. Modern models reason over structured text directly.
<50ms
Sub-Agents
Explore, Plan, general-purpose. Parallel agents for research, multi-step tasks.
on-demand
Hooks
Sparingly used. Only when deterministic enforcement is required (PostToolUse on writes, settings updates).
deterministic
MCP servers
15+ adapters. Notion, Drive, Gmail, Calendar, Computer-Use, Chrome, Telegram. All optional, loaded per session.
plug-and-play
Mobile interface
Telegram bot
AWS Lightsail Frankfurt. Voice + text triggers. Calls back to Claude API with brain context.
3.71.254.207

What the harness does: Loop, files, context, safety. Four things only. Push intelligence up into Skills, push execution down into MCP and Bash, keep the harness thin.
