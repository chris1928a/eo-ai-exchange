# Skills

Runnable Claude Code Skills, contributed by the EO AI Productivity Exchange community.

Each skill follows the [Claude Code Skills spec](https://docs.claude.com/skills) with:
- `SKILL.md` (description, when-to-trigger, instructions)
- Optional `examples/` and `scripts/` folders

## How to add a skill

See [CONTRIBUTING.md](../CONTRIBUTING.md). PRs welcome from the community.
