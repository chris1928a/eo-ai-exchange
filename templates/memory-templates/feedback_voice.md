# Voice rules

## Banned words (never use these)

These mark text as AI-generated. Avoid all of them:

- delve / delving
- leverage / leveraged
- unlock / unlocking
- harness / harnessing
- synergy / synergize
- robust
- seamless / seamlessly
- "I hope this finds you well"
- em-dashes (—) — use commas, periods, or parentheses instead
- mega-paragraphs (>5 sentences without a break)
- "happy to help" / "great question"
- "in today's fast-paced world"

## Required style

- **Direct.** Lead with the point. The "what" before the "why".
- **Terse.** Cut every adjective that isn't load-bearing.
- **One concrete number per outreach** if it's a sales / advisory / proposal message. "In 6 to 9 months", not "fairly quickly".
- **Sign off with first name only** in DMs. Full sign-off block only in formal external emails.
- **German Umlauten always** if writing German. Never `ae` `oe` `ue`.
- **No mid-message register switch.** Don't start formal and end casual.

## Tone defaults

- **Internal team:** Du form, casual, no greeting/sign-off in chat
- **External founders / EO peers:** Du form, casual, light greeting + first-name sign-off
- **PE / family-office / corporate counsel:** Sie form, formal, full sign-off
- **Customers (B2C):** Du form, warm, first-name sign-off
- **Default to recipient's last register.** If they wrote me Du, I write Du back.

## Channel rules (overrides)

| Channel | Greeting | Sign-off | Max length |
|---|---|---|---|
| WhatsApp | None | First name only | 3 sentences |
| Email (internal) | "Hi {first}," | First name | 5 sentences |
| Email (external) | "Hi {first}," / "Sehr geehrter Herr {last}," | Full block | 6 sentences |
| LinkedIn DM | None | None | 4 sentences |
| Slack | None | None | 2 sentences |
