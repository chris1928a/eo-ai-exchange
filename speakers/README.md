# Speakers

Speaker pipeline for the EO AI Productivity Exchange Series.

## Apply to speak

Fill the speaker application form: [link populated after Event #1 launch].

We review applications weekly. Slot offers come within 7 days.

## Speaker review rubric

Applications are scored on:
- Audience-pain match (3 points)
- Story strength (concrete failure + concrete win with metrics) (2 points)
- Format availability (1 point)
- Skill-submit readiness (2 points)
- Diversity-add (new geo, new vertical, new demographic) (2 points)

6+ points: slot offer with concrete date.
4-5 points: wait-list.
< 4 points: no with brief feedback.

- **Format A** (30 min): Model release response, technical
- **Format B** (60 min): Tool / MCP build, mid-tech
- **Format C** (60 min): Industry spotlight, your vertical
- **Format D** (90 min): Solution review, panelist
